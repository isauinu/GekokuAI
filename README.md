# GekokuAI
Status:
Alpha 0.1 (Early Development)

a self-written local LLM Launcher directly from your CLI alone.
currently supports all linux distro that uses APT, pacman, and DNF as its package manager
## Installation

- `git clone https://github.com/isauinu/GekokuAI/`
- `cd GekokuAI`
- `chmod +x setup.sh`
- `./setup.sh`

## Commands
- gekoku serve [MODEL]
- gekoku pull [REPO]
- gekoku status
- gekoku info
- gekoku remove [MODEL]
- gekoku doctor
- gekoku list
- gekoku stop
