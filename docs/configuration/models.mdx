# Models

Configuring models and changing how they run can be done the same way as changing `config.toml`. 
However, to edit your preffered models, they're installed located in `$HOME/.gekokuai/models` in which each models are seperated in their own `.toml` file and you can edit them with your preffered text editor like `nano`
