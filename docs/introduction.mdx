# Introduction

## GekokuAI

GekokuAI is a versatile local LLM Orchestration tool that simplifies downloading, managing, and serving LLM Models, it's built ontop llama.cpp while providing centralized CLI, runtime managment, and hardware based optimizations


GekokuAI is still in very early development, expect things to change everytime. I welcome every contributions from anyone that wants to help me with this project!

## Features

<Steps>
  <Step title="Run multiple models">
    Directly run your local LLM models at the same time with minimal efforts
  </Step>
  <Step title="Download your models out of the box">
    Use `gekoku pull` and start downloading your models of choice and use it
  </Step>
  <Step title="Compatible with any workflow">
    Gekoku uses OpenAI Compatible API which allows you to serve your models to any workflow like Open-webui
  </Step>
</Steps>
